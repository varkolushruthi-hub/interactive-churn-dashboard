# Task 5 - Interactive Dashboard Development

## 📌 Internship
VirtualWorks Lab by Emogi — Data Analytics Internship

## 📝 Task Overview
Created an interactive dashboard that presents key insights clearly
and effectively. Used appropriate charts and visual elements to
communicate trends, performance metrics, and actionable insights
for decision-making.

## 🛠️ Tools Used
| Tool | Purpose |
|---|---|
| Python 3 | Core programming |
| Dash | Interactive web dashboard framework |
| Plotly | Interactive charts |
| Pandas | Data processing |
| Jupyter Notebook | Development |

## 📊 Dashboard Features
- 🔽 Dropdown filter by Subscription Plan
- 👥 KPI Cards: Total Customers, Churned, Churn Rate, Avg Tenure
- 🥧 Churn Distribution donut chart
- 📊 Churn by Subscription Plan bar chart
- 📊 Churn by Payment Delay bar chart
- 📦 Login Frequency vs Churn boxplot
- 📈 Tenure Distribution histogram
